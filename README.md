# Neetchi (நீட்சி) - Download Center

Financial Analytics Extension Application

## Download Latest Version

Go to [Releases](https://github.com/Ganathipan/neetchi-releases/releases) 
to download the latest installer.

## Installation

1. Download `Neetchi_Setup_v*.*.*.exe`
2. Run the installer
3. Follow the setup wizard
4. Launch Neetchi from Start Menu or Desktop

## Auto-Updates

The application checks for updates automatically. 
When a new version is available, you'll be notified within the app.

## System Requirements

- Windows 10/11 (64-bit)
- 500 MB free disk space
- Internet connection (for updates)
